# Employee Management System

## 📌 Project Overview

Employee Management System is a full-stack MERN application developed to manage employee records efficiently. The application allows users to add, view, and delete employee details through a user-friendly interface while storing data securely in MongoDB.

---

## 🚀 Features

* Add New Employee
* View Employee Records
* Delete Employee Records
* REST API Integration
* MongoDB Database Connectivity
* Responsive User Interface
* Full Stack MERN Architecture

---

## 🛠️ Tech Stack

### Frontend

* CSS3

### Backend

* Node.js

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone YOUR_REPOSITORY_LINK
```

### Backend Setup

```bash
cd backend

npm install

npm run dev
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

## 🔗 API Endpoints

### Get All Employees

```http
GET /employees
```

### Add Employee

```http
POST /employees
```

### Update Employee

```http
PUT /employees/:id
```

### Delete Employee

```http
DELETE /employees/:id
```
